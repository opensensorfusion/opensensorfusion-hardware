# OSF GREEN board photos

Place real OSF GREEN board photos in this directory.

Recommended filenames:

- `OSF_GREEN_top.jpg`
- `OSF_GREEN_bottom.jpg`
- `OSF_GREEN_with_raspberry_pi.jpg`

Photos should show the actual board state clearly. Avoid edits that hide board
markings, connectors, or component placement when the image is used for hardware
identity documentation.
