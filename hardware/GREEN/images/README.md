# OSF GREEN images

This directory contains board-specific media for OSF GREEN.

Use these subdirectories:

- `photos/` for real board photos.
- `pcb-2d/` for 2D PCB views.
- `renders-3d/` for 3D board renders.

## Recommended filenames

Board photos:

- `photos/OSF_GREEN_top.jpg`
- `photos/OSF_GREEN_bottom.jpg`
- `photos/OSF_GREEN_with_raspberry_pi.jpg`

2D PCB views:

- `pcb-2d/OSF_GREEN_2d_top.png`
- `pcb-2d/OSF_GREEN_2d_bottom.png`

3D renders:

- `renders-3d/OSF_GREEN_3d_angle90.png`
- `renders-3d/OSF_GREEN_3d_angle30.png`

Keep image names stable when they are referenced by README files or external
project pages.
