# Open Hardware Release Checklist

OSHWA certification is not claimed for OSF GREEN.

This repository is being organized so the hardware release can be reviewed as
an open hardware project. The following items should be published before any
certification claim is made.

## Checklist

- schematic source
- schematic PDF export
- PCB source
- Gerber or fabrication outputs
- bill of materials
- assembly notes
- firmware source location
- firmware license
- documentation license
- board photo
- attribution and trademark notes

## Notes

Certification status should not be implied by the repository name or README.
When certification work starts, record the exact application state and the
public hardware release version that it covers.
